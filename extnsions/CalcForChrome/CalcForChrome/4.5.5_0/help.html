Keyboard Shortcuts:

On Mac computers the CMD key works in the place of the CTRL key.

  • Copy (CTRL + C)
  • Paste (CTRL + V)
  • Divide (/)

  • Minus (-)
  • Plus (+)
  • Equals (= or Enter)
  • +/- (F9 or R)
  • Square Root (SHIFT + @)
  • Open Bracket (SHIFT + 9)
  • Close Bracket (SHIFT + 0)

  • AC (Home)
  • Clear Memory (CTRL + L)
  • Recall Memory (CTRL + R)
  • Set Memory (CTRL + M)
  • Add Memory (CTRL + P)

  • Open Info (F11)
